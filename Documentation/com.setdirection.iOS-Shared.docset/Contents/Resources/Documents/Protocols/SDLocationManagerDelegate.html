SDLocationManagerDelegate Protocol Reference

Conforms to NSObject
Declared in SDLocationManager.h

Overview

Allows the SDLocationManager to persistently record whether it has received at least 1 valid location update during the life of the app. This is useful when other classes need to know whether user has tapped ‘OK’ on the iOS location permission alert.
