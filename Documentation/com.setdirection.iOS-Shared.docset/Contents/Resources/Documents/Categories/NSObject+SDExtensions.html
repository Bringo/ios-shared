NSObject(SDExtensions) Category Reference

Declared in NSObject+SDExtensions.h

Overview

A number of extensions to the base NSObject class. Most of these came into existence to make the loading of nibs much easier.

Instance Methods

callSelector:returnAddress:argumentAddresses:

Invoke arbitrary selectors on the receiver with arbitrary arguments and with a place to store the result of the invocation.

- (void)callSelector:(SEL)aSelector returnAddress:(void *)result argumentAddresses:(void *)arg1, ...

Parameters

aSelector

The selector to invoke. It must exist on the receiver.

result

A pointer to the location where the result of the invocation can be stored. Optional.

arg1,...

The ordered list of arguments to pass to the selector being invoked.
